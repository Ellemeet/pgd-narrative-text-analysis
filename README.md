This repository contains the negative affect word list from the Opinion Lexicon developed by Hu & Liu (2004).

## Description
The lexicon consists of approximately 4600 English words and 3000 Dutch words expressing negative sentiment.
It is commonly used in sentiment analysis and computational linguistics research.

## Source
Hu, M., & Liu, B. (2004).   *Mining and Summarizing Customer Reviews.*  
Proceedings of the ACM SIGKDD International Conference on Knowledge Discovery and Data Mining.

## Translation
English words were manually translated by the researcher and consolidated into 2,957 Dutch negative affect words, removing semantic overlap and redundancy.

## Usage
This dataset is provided for non-commercial academic research purposes. Please cite the original authors when using this lexicon.
